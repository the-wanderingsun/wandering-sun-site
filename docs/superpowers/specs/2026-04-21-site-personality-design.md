# 个人网站个性化升级 · 设计文档

**日期：** 2026-04-21
**作者：** 梅子林（太阳在世界游荡）

---

## 用户偏好（已确认）

| 维度 | 选择 |
|------|------|
| 互动感 | 有动效和互动（1D）|
| 情感基调 | 有温度，像真实的人（2B）|
| 视觉参考 | Notion 的克制感 |
| 现有配色 | 暖橙色调（保留）|

---

## 目标

让网站从"功能完整但平淡"升级为"有个性、有温度、让人想多停留一会儿"。  
每一个改动都应该服务于一个目的：**让访客感受到这是一个真实的人在认真经营的地方**。

---

## 范围

### 一、首页（`app/page.tsx`）— 重点改造

#### 1.1 打字机效果（Typewriter）
- 介绍语逐字出现，模拟实时打字
- 打完后光标持续闪烁
- 实现：纯 CSS + JS，无需额外库

#### 1.2 「现在」小模块（Now Widget）
- 位置：社交链接下方，页面中部
- 内容：手动维护的 3 个状态
  - 🎵 在听：（歌手 · 专辑名）
  - 📖 在读：（书名 · 作者）
  - 🌍 在：（当前城市）
- 数据来源：`public/data/now.json`（手动更新，push 即生效）
- 样式：小卡片，Notion 风格，带左侧色条

#### 1.3 随机一句话（Random Quote）
- 每次刷新显示一句你喜欢的话
- 数据来源：`public/data/quotes.json`（你自己维护，想加就加）
- 位置：首页底部，灰色小字
- 样式：居中斜体，像书页底部的题词

#### 1.4 卡片悬停动效
- 现有的三个导航卡片（每日日报、GitHub、播客）加悬停微上浮
- `hover:-translate-y-1 hover:shadow-md transition-all duration-200`

---

### 二、侧边栏（`components/Nav.tsx`）— 小优化

#### 2.1 「现在在」状态
- 侧边栏底部加一行实时状态，显示当前城市
- 数据读自 `public/data/now.json`

#### 2.2 社交链接图标
- 侧边栏底部加 Twitter / 小红书图标链接（SVG）

---

### 三、博客页面（`app/blog/`）— 增强体验

#### 3.1 阅读时长估算
- 列表页每篇文章显示「约 X 分钟」
- 计算：字数 / 300（中文阅读速度）

#### 3.2 文章封面色块
- 每篇文章根据 frontmatter 的 `color` 字段显示左侧色条
- 没有 color 字段则随机从暖色调中取一个
- 让列表页有视觉层次，不单调

---

### 四、推荐页面（`app/links/page.tsx`）— 新增

#### 4.1 页面定位
- 收录你认为值得分享的网站、工具、内容
- 不是导航站，是"你的书签"，有你的评语

#### 4.2 数据结构（`public/data/links.json`）
```json
[
  {
    "category": "值得一读",
    "items": [
      {
        "name": "Wait But Why",
        "url": "https://waitbutwhy.com",
        "desc": "用超长漫画解释复杂概念，Elon Musk 最爱的博客",
        "emoji": "🐢"
      }
    ]
  }
]
```

#### 4.3 样式
- 按分类分组，每个分类用小标题隔开
- 每个链接：emoji + 名字 + 一句话描述 + 外链箭头
- Notion 风格，干净

#### 4.4 导航入口
- 侧边栏加「推荐」链接（🔖）

---

### 五、全局细节

#### 5.1 页面切换过渡动画
- 使用 Next.js 的 `layout` 层加 `animate-fade-in`
- 进入页面时内容淡入（0.2s），不突兀

#### 5.2 背景纹理（可选）
- 在 `#fdf8f3` 基础上加极淡的噪点纹理（CSS grain）
- 让背景有纸张质感，而不是纯色块

---

## 不做的事（YAGNI）

- ❌ 3D 效果 / WebGL（太重，不符合 Notion 克制感）
- ❌ 暗色模式（破坏暖色调品牌感，日后再说）
- ❌ 鼠标光晕跟随（移动端无效，实现成本高）
- ❌ 访客计数器（需要后端，暂不引入）
